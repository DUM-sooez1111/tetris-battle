# Tetris Battle

같은 와이파이에 있는 친구와 방을 만들고 참가해서 2-5명이 함께 플레이할 수 있는 테트리스 웹 게임입니다.

## GitHub Pages에서 실행할 때

GitHub Pages는 정적 웹사이트만 실행합니다. 그래서 `server.js`는 GitHub Pages에서 실행되지 않습니다.

GitHub Pages 주소에서는 아래 기능을 사용할 수 있습니다.

- 1인 플레이
- 모바일 조작 UI
- 상점과 인벤토리
- 코인, 블록 스킨, 배경 테마, AI 노래
- 설정, 음악, AI 난이도 화면

GitHub Pages 주소에서는 아래 기능은 로컬 서버가 없어서 제한됩니다.

- 같은 와이파이 멀티플레이
- 방 만들기/방 참가의 실시간 동기화
- 관전 실시간 동기화

친구와 같이 멀티플레이를 하려면 GitHub Pages가 아니라 방장 컴퓨터에서 아래처럼 실행해야 합니다.

```bash
npm start
```

그 다음 친구는 방장 컴퓨터의 `http://192.168.x.x:5173` 주소로 접속합니다.

## 1. 방장 컴퓨터에서 실행하는 방법

처음 한 번만 설치합니다.

```bash
npm install
```

게임을 켭니다.

```bash
npm start
```

또는 아래 명령어도 똑같이 사용할 수 있습니다.

```bash
npm run dev
```

실행하면 터미널에 접속 주소가 표시됩니다.

```text
내 컴퓨터 접속: http://localhost:5173
친구 접속 주소:
  http://192.168.x.x:5173
```

방장 본인은 `http://localhost:5173`으로 접속하면 됩니다.

## 2. 내 컴퓨터 IP 주소 확인하는 방법

게임을 실행하면 터미널에 `친구 접속 주소`가 자동으로 표시됩니다. 그 주소를 친구에게 보내면 가장 쉽습니다.

직접 확인하고 싶다면:

Mac에서는 터미널에 아래 명령어를 입력합니다.

```bash
ipconfig getifaddr en0
```

Windows에서는 명령 프롬프트에 아래 명령어를 입력합니다.

```bat
ipconfig
```

`IPv4 주소`처럼 보이는 `192.168.x.x` 값을 찾으면 됩니다.

## 3. 친구 컴퓨터에서 접속하는 방법

친구는 방장 컴퓨터의 IP 주소와 포트 번호 `5173`을 붙여서 접속합니다.

예를 들어 방장 IP가 `192.168.0.23`이면 친구는 아래 주소를 엽니다.

```text
http://192.168.0.23:5173
```

주의: 친구 컴퓨터에서는 `localhost`로 접속하면 안 됩니다. `localhost`는 자기 컴퓨터라는 뜻입니다.

## 4. 방 만들기와 참가 방법

방장:

1. 닉네임을 입력합니다.
2. `2인 대전`, `3인 대전`, `4인 대전`, `5인 대전` 중 하나를 선택합니다.
3. `방 만들기`를 누릅니다.
4. 로비 왼쪽 아래의 `SHARE LINK` 또는 방 코드를 친구에게 보냅니다.
5. 모두 들어오고 `READY`를 누르면 `START GAME`을 누릅니다.

친구:

1. 방장이 알려준 주소로 접속합니다.
2. 닉네임을 입력합니다.
3. 공유 링크로 들어온 경우 방 코드가 자동으로 채워집니다.
4. 직접 들어온 경우 `방 참가하기`를 누르고 방 코드를 입력합니다.
5. `READY`를 누릅니다.

## 5. 접속이 안 될 때 확인할 것

- 방장과 친구가 같은 와이파이에 연결되어 있는지 확인합니다.
- 방장 컴퓨터에서 방화벽 허용 알림이 뜨면 허용합니다.
- 친구가 `localhost`가 아니라 `192.168.x.x` 주소로 접속했는지 확인합니다.
- 포트 번호가 `5173`인지 확인합니다.
- 방장 컴퓨터에서 `npm start` 또는 `npm run dev`가 계속 실행 중인지 확인합니다.

## 플레이 중 보이는 정보

각 플레이어는 자기 화면에서 큰 테트리스 보드를 플레이합니다.

상대는 화면 아래쪽 작은 카드로 표시되며 아래 정보가 실시간으로 보입니다.

- 닉네임
- 점수
- 레벨
- 삭제한 줄 수
- 생존 여부 또는 게임 오버 여부
- 현재 보드 상태

2인전에서는 한 명이 게임 오버되어도 다른 한 명은 계속 플레이합니다. 두 명 모두 게임 오버되면 점수가 높은 사람이 승리합니다.

3인/4인/5인전에서는 마지막 1명이 남을 때까지 계속 진행됩니다.
